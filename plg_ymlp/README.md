jForms YMLP Plugin
======
A jForms plugin to save submitted data to a YMLP mailing list.

### Instructions

1. Install the plugin in Joomla, then configure & publish it in your plugin manager
2. Open the jForms module which you wish to use in your module manager, click on the options tab and scroll to the bottom. 
You will see a new set of options called "Your Mailing List Provider"
